# 									日志埋点模型



## **启动埋点**

<!--表中字段如无法取得或为空值，直接传空字符串即可-->

| 04   | A111111111111_IOS-default                                    | 格式：后台创建的应用ID_应用平台-workspaceId                  |
| 05   | 1.0.1                                                        | 客户端版本，即应用版本                                       |
| 06   | d5557b75-ff80-4aab-86a6-9b1a522bbbce                         | 会话ID                                                       |

| 09   | 1                                                            | 是否首次启动：0 首次，1 非首次                               |

| 11   | 4.4.2                                                        | 操作系统版本                                                 |
| 12   | WIFI                                                         | 网络类型，如： WIFI、2G、3G、4G                              |

| 14   | goose                                                        | 渠道号                                                       |
| 15   | Wn1111111111111111111QxL                                     | 设备 ID                                                      |
